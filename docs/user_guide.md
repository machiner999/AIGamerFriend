# AI Gamer Friend 使い方ガイド

ゲーム画面にカメラを向けるだけで、AIの友達「ユウ」が音声でリアクションしてくれるアプリです。一人でゲームをしていても、隣に友達がいるような感覚で楽しめます。

## 必要な環境

- Android 8.0 以上のスマートフォン
- インターネット接続（Wi-Fi推奨）
- カメラとマイクの使用許可

## はじめかた

### 1. アプリを起動する

初回起動時、カメラとマイクの使用許可を求められます。「許可する」をタップしてください。

> もし許可し忘れた場合は、端末の「設定」→「アプリ」→「AIGamerFriend」→「権限」からカメラとマイクを許可できます。

### 2. オンボーディング画面

初回起動時に案内画面が表示されます。ユウの表情プレビューが表示されるので、雰囲気を確認したら「はじめる」ボタンをタップして閉じてください。

## 基本的な使い方

画面下部の「▶ 開始」ボタンをタップすると、ユウとのセッションが始まります。

> 設定で「自動スタート」をオンにすると、アプリ起動時に自動でセッションが開始されます。

スマートフォンの背面カメラをTVやモニターのゲーム画面に向けてください。ユウがゲーム画面を見て、音声でリアクションしてくれます。

マイクが有効な状態なら、ユウに話しかけることもできます。ゲームの感想を言ったり、質問したりしてみましょう。

ユウはゲームに関する情報をWEB検索で調べることもできます。攻略法やキャラクターの情報など、気になることを聞いてみてください。

ミュートボタン（マイクのアイコン）をタップすると、マイクを一時的にオフにできます。ミュート中はボタンが赤く表示されます。

### 音声コマンド（ハンズフリー操作）

| 言葉の例 | 操作 |
|---------|------|
| 「終わり」「やめる」「ストップ」 | セッション終了 |
| 「ミュート」「黙って」「静かにして」 | ミュート切替 |

> ゲームの文脈での「終わり」には反応しません。ミュート中は音声が届かないため、ミュート解除は画面タッチで行ってください。

「■ 終了」ボタンをタップするか、「終わり」「ストップ」などと声をかけるとセッションが終了します。

## 画面の見かた

| 場所 | 表示内容 |
|------|---------|
| 左上 | 接続状態（🟢 LIVE / 🟡 接続中 / 🟡 再接続中 / 🔴 エラー）|
| 左上の下 | 🎮 検出されたゲーム名（検出時のみ、スライドインで表示）|
| 中央下 | ユウの顔（感情でカラーと表情が変化、音声に反応するリング付き）|
| 最下部 | コントロールパネル（音声レベルバー + 各種ボタン）|

### 接続状態について

- **LIVE**（緑の点滅）: 正常に接続中。ユウがゲーム画面を見ています
- **応答待機中**（橙の高速点滅）: 接続中だがユウの応答に時間がかかっています
- **接続中 / 再接続中**（黄）: サーバーに接続しています。しばらくお待ちください
- **エラー**（赤 + シェイク）: 接続に失敗しました。通信環境を確認してください

## ユウの表情について

ユウはゲームの展開に応じて7つの表情を見せてくれます。感情ごとに顔のカラーが変わり、変化時にはパーティクルエフェクトが表示されます。

| 表情 | カラー | どんなとき |
|------|--------|-----------|
| ふつう | 緑 | 落ち着いている状態 |
| うれしい | 緑 | 楽しいシーンや良いプレイのとき |
| 大興奮 | 金 | すごいプレイや盛り上がる場面で |
| おどろき | シアン | 予想外の展開に驚いたとき |
| 考え中 | 紫 | 状況を分析しているとき |
| 心配 | 橙 | ピンチの場面や不安なとき |
| かなしい | 青 | 残念な結果や負けたとき |

ユウの顔の周りにある破線リングは、音声レベルに応じて明るさやパターンが変化します。また、定期的にまばたきをしたり、目線を動かしたりするアイドルアニメーションがあります。

## 設定

画面下部の歯車アイコンをタップすると設定画面が開きます。

### 声の種類
AOEDE（デフォルト）、KORE、PUCK、CHARON、FENNIRの5種類から選べます。

### リアクションの強さ
おとなしめ / ふつう（デフォルト）/ テンション高めの3段階。

### 自動スタート
オンにすると、アプリ起動時に自動でセッションが開始されます。

### 記憶のクリア
過去のセッションでユウが覚えた内容をすべて削除できます。

## 自動再接続について

長時間使っていると、通信が一時的に切れることがあります。通常は自動で再接続されるので、そのままお待ちください。再接続中はステータスが「再接続中」に変わり、完了すると「LIVE」に戻ります。

## トラブルシューティング

### 「エラー」と表示される・接続できない

- インターネット接続を確認してください（Wi-Fi推奨）
- 「▶ 開始」ボタンを再度タップしてみてください
- アプリを再起動してみてください

### ユウが反応しない

- ミュートになっていないか確認してください（マイクアイコンが赤くなっていないか）
- 背面カメラがゲーム画面をしっかり映しているか確認してください
- セッションが開始されているか（LIVE表示）確認してください

### 音が聞こえない

- 端末の音量が十分か確認してください
- セッションがLIVE状態か確認してください
